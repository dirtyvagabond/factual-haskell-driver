join ∷ [a] → [[a]] → [a]

The join function joins a list of lists into a list using a separator list. The most common use case is for joining Strings with a common separator String.

joinAndFilter ∷ [String] → String

This function filters out empty Strings from a list before joining the Strings with an & character. The use case is forming query path Strings.
